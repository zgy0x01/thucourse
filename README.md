# THUCourse：THU 选课社区

基于jCourse：SJTU 选课社区源代码开发

## 开始使用

使用 [Next.js](https://nextjs.org/) 框架, UI 库选用 [Ant Design](https://github.com/ant-design/ant-design/)。

安装依赖

```bash
$ yarn
```

本地调试

```bash
$ yarn dev
```

构建

```bash
$ yarn build
```
